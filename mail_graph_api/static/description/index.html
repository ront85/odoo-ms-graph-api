Microsoft Graph API Mail Gateway

Send emails via Microsoft Graph API instead of SMTP

Overview

This module replaces the standard SMTP mail gateway with Microsoft Graph API integration for sending emails from Odoo. It allows for a more secure and reliable email delivery through Microsoft 365.

Key Features

Microsoft Graph API Integration

Send emails via Graph API instead of SMTP.

Fully Configurable from Odoo UI

Users can enter Azure credentials in the Outgoing Mail Servers configuration.

OAuth2 Authentication

Secure API calls without storing plain-text passwords.

Multi-Tenant Support

Works with multiple Office 365 accounts.

Fallback to SMTP

If Graph API fails, optionally fall back to SMTP.

Logging & Error Handling

Capture detailed logs for debugging failed email sends.

Configuration

Setting up the Microsoft Graph API Mail Gateway is quick and easy:

  1. Install the module from the Odoo App Store.
  2. Create an Azure AD application with Mail.Send permissions.
  3. Configure your Outgoing Mail Server with your Azure credentials.
  4. Enable Microsoft Graph API on your mail server.
  5. Authenticate with Microsoft OAuth2 flow.
